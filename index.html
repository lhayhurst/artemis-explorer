Artemis Preset Explorer — Guide

Bank Tabs

Eight bank tabs (A–H) run across the top. Click any tab to switch to it.

IndicatorMeaning
Green dotBank is loaded
Cyan F badgeFactory preset bank
Green U badgeUser-loaded bank
Orange ● badgeBank has unsaved edits

Loading Presets

  • Load .syx — click the button or drag & drop one or more .syx files onto the window. Files named with a bank letter (e.g. bank-b.syx) land in that bank; others go into the first empty slot.
  • Load Factory — fetches the Dreadbox factory bank for whichever tab is active.
  • MIDI receive — see the MIDI section below.

Preset Sidebar

  • Filter box — type to filter by name or play mode.
  • Left-click a preset to select it and open its detail view.
  • Right-click a preset to add it to the compare selection (up to 3).
  • Mini bars show cutoff, resonance, attack, and release at a glance.
  • An orange ● after a preset name means it has been edited but not yet sent or exported.

Detail View

Shows all parameters for the selected preset in collapsible cards: VCO, VCF (LP/HP), Envelopes, LFOs, FX (Distortion, Modulation, Delay, Reverb), Seq/Arp, and Global. Click a card header to collapse or expand it.

Modulation Matrix — tabs below the cards show per-source routing (Mod Wheel, Velocity, Aftertouch, CC74, Key Track). An orange dot means that source has active routing.

✎ Edit / Done — click Edit to enter edit mode. Float parameters become sliders, enums become dropdowns, booleans become toggle buttons, and the preset name becomes a text field. Click Done to return to read-only view. Edits are in-memory only until you send or export.

Table View

All 64 presets in a sortable table. Click a column header to sort; click again to reverse. Use the mode filter chips to narrow the list.

Compare View

Right-click 2 or 3 presets in the sidebar, then switch to Compare. Parameters that differ between presets are highlighted in orange.

MIDI Bar

Requires Chrome or Edge with SysEx access granted. The MIDI bar is the strip fixed to the bottom of the window.

IN side — select your Artemis input port, then click Listen. Trigger a preset or bank export from the Artemis front panel; the app loads the data automatically. Click Stop to stop listening, or Clear to discard buffered data.

Single preset received — loads into the currently selected bank slot. An orange ● appears next to the slot if it already had edits; receiving clears it.

Bank dump received — all 64 presets load into the identified bank, or the first free bank if unidentified.

OUT side — select your Artemis output port, then use Send Preset or Send Bank.

Send Preset — loads the preset into Artemis temp memory for audition. Save from the front panel if you want to keep it.

Send Bank — sends all 64 presets. The Artemis prompts you to choose a destination bank slot.

The coloured dot next to each section label turns green when a port is selected and connected.

Export

Download All exports every loaded bank as a .zip containing one .syx file per bank, including any in-memory edits.
